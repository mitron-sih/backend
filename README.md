# Backend

Django based backend server to manage all the functionalities of the website. 

Implemented the following endpoints
- /user : To add, remove and update a user into the database
- /schemes : To get information about the various Government schemes
- /volunteer : To add, remove and update  a volunteer into the database
- /aids : To add, remove and update aids and assistive devices into the database
- /userLogin: Implemented User Authentication mechanism
- /volunteerLogin: Implemented Volunteer Authentication mechanism

## Dependencies
- Django Rest Framework
- Google Translate API
- SQL 

( requirements.txt is added to install all the dependencies )
